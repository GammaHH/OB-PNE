- 結構均含有 Thioamide (S=C-N)活性必須結構
- 副作用均有 顆粒性白血球缺乏（Agranulocytosis）
- 藥物 Onset 慢 ，最少須四周

| 項目    | PTU(Propylthiouracil) | MMI(Methimazole) 媽祖 |
| ----- | --------------------- | ------------------- |
| 半衰期   | 短                     | 長                   |
| 作用強度  | 弱                     | 強                   |
| onset | 快(因為多抑制一條路徑)          | 慢                   |
| 懷孕    | 前期可用                  | 後期可用                |
| 副作用   | +肝毒性(因胎兒肝在後期成熟)       | +再生不良性貧血            |
